# Araç-Filo-kontrol

📌 Proje Özeti

Bu proje, araç kiralama süreçlerini uçtan uca dijitalleştiren bir otomasyon sistemidir.

Sistem şu adımlardan oluşur:

Araç verileri Google Sheets üzerinden API olarak yayınlanır

HTML tabanlı bir web arayüzü ile araçlar listelenir

Kullanıcı araçları seçer

Seçilen araçlar Google Forms’a otomatik aktarılır

Kullanıcı sözleşme bilgilerini girer

Form submit edildiğinde Apps Script tetiklenir

Araç takip tablosu otomatik güncellenir
